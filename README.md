## Course Structure

### Part 1: Computer Fundamentals

| Module | Title | Topics | Hours |
|--------|-------|--------|-------|
| 1 | Binary & Digital Logic | Number systems, Binary arithmetic, Boolean algebra | 8 |
| 2 | Transistors & CMOS | MOS transistors, CMOS logic, Power consumption | 10 |
| 3 | Logic Gates & Boolean Algebra | AND, OR, NOT, NAND, NOR, XOR, Karnaugh maps | 12 |
| 4 | Combinational Circuits | Multiplexers, Decoders, Adders, ALU design | 14 |
| 5 | Sequential Circuits & FSM | Flip-flops, Registers, State machines | 12 |

### Part 2: CPU Architecture

| Module | Title | Topics | Hours |
|--------|-------|--------|-------|
| 6 | CPU Architecture & Datapath | Von Neumann, Harvard, Instruction sets, Pipelining | 16 |
| 7 | Memory Hierarchy & Cache | Cache design, Virtual memory, Page tables | 10 |

### Part 3: C Programming

| Module | Title | Topics | Hours |
|--------|-------|--------|-------|
| 8 | C Programming Fundamentals | Variables, Control flow, Functions, Arrays | 12 |
| 9 | Advanced C Programming | Pointers, Dynamic memory, Structures, File I/O | 16 |

### Part 4: C++ Programming

| Module | Title | Topics | Hours |
|--------|-------|--------|-------|
| 10 | C++ Object-Oriented Programming | Classes, Inheritance, Polymorphism, Encapsulation | 14 |
| 11 | Modern C++ (C++11/14/17/20) | Auto, Lambdas, Smart pointers, Move semantics, STL | 18 |
| 12 | Advanced Systems Programming | Templates, Concurrency, RAII, Performance optimization | 20 |

**Total: 12 Modules | 120+ Hours | 500+ Diagrams | 500+ Code Examples**

## Installation

```bash
# Clone the repository
git clone https://github.com/shashankdakshiitb/cs-mastery-course.git

# Navigate to the directory
cd cs-mastery-course

# Install dependencies
npm install

# Start the development server
npm run dev
```
